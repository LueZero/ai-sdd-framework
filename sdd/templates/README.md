# SDD 範本

此目錄包含 SDD 流程中使用的文件範本。

## 範本清單

| 範本 | 用途 | 使用階段 |
|------|------|---------|
| [feature-spec.tmpl.md](feature-spec.tmpl.md) | 功能規格書 | Phase 2: 規格化 |
| [api-contract.tmpl.md](api-contract.tmpl.md) | API 契約定義 | Phase 3: 設計 |
| [component-spec.tmpl.md](component-spec.tmpl.md) | 元件規格 | Phase 3: 設計 |
| [data-model.tmpl.md](data-model.tmpl.md) | 資料模型設計 | Phase 3: 設計 |
| [system-design.tmpl.md](system-design.tmpl.md) | 系統設計文件 | Phase 3: 設計 |
| [adr.tmpl.md](adr.tmpl.md) | 架構決策記錄 | Phase 3: 設計 |
| [test-plan.tmpl.md](test-plan.tmpl.md) | 測試計劃 | Phase 5: 測試 |

## 使用方式

1. 複製範本到對應的 `docs/sdd/` 子目錄
2. 依照範本中的提示填寫內容
3. 或使用對應的 Prompt 讓 AI 協助填寫

## 客製化

你可以根據專案需求修改範本或新增新範本。建議：
- 保持範本的結構化格式
- 每個區塊提供填寫提示
- 包含品質檢查清單
